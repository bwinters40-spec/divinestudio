## How to Deploy (Fresh Start)

### Step 1 — Create a GitHub Repo
1. Go to github.com and sign in (create a free account if needed)
2. Click the **+** icon → **New repository**
3. Name it: `divine-studio`
4. Set to **Public**
5. Check **"Add a README file"**
6. Click **Create repository**

### Step 2 — Upload Your Files
1. In your new repo, click **Add file → Upload files**
2. Drag and drop ALL files from this ZIP:
   - `index.html`
   - `vercel.json`
   - `.gitignore`
   - `README.md`
3. Click **Commit changes**

### Step 3 — Deploy on Vercel
1. Go to vercel.com → sign in with your GitHub account
2. Click **Add New → Project**
3. Select your `divine-studio` repository
4. Click **Deploy** (leave all settings as default)
5. Done — your site is live in ~60 seconds

### Step 4 — Connect Your Domain (divinestudio.beauty)
1. In Vercel, go to your project → **Settings → Domains**
2. Type `divinestudio.beauty` and click **Add**
3. Vercel will show you DNS records to add
4. Log into Namecheap (or wherever you bought the domain)
5. Go to **Advanced DNS** and add the records Vercel gives you
6. Wait 10–30 minutes for it to go live

## How to Update the Site
1. Go to your GitHub repo
2. Click `index.html`
3. Click the **pencil/edit icon**
4. Make your changes
5. Click **Commit changes**
6. Vercel auto-deploys in ~60 seconds
